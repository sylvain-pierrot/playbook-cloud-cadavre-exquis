<h3 align="center">Playbook to run Cluster Multi-Machine K3s</h3>

## Application accessible locally on the ip:

*192.168.50.10*

## Deploy Cluster:

```bash
bash deployClusterKube.sh
```

## Destroy cluster:

```bash
bash destroyClusterKube.sh
```
